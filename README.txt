PROJECT TITLE:
Password Strength Analyzer

DESCRIPTION:
This project checks the strength of user-entered passwords.

FEATURES:
- Password length check
- Uppercase/lowercase detection
- Number detection
- Special symbol detection
- Common weak password detection
- Strong password suggestion
- SHA-256 hashing
- SQLite database integration

TECHNOLOGIES USED:
- Python
- SQLite
- hashlib
- Regular Expressions

HOW TO RUN:
1. Open terminal
2. Run:
   py password_analyzer.py

OUTPUT:
The system displays:
- Password strength
- Suggestions
- Strong password recommendation
- Hashed password
