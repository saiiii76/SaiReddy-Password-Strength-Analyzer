# Sai Reddy Password Strength Analyzer

## PROJECT TITLE
Password Strength Analyzer

## DESCRIPTION
This project checks the strength of user-entered passwords using cybersecurity best practices and provides recommendations for creating secure passwords.

## FEATURES
- Password length check
- Uppercase and lowercase detection
- Number detection
- Special symbol detection
- Common weak password detection
- Strong password suggestions
- SHA-256 password hashing
- SQLite database integration
- Flask-based web interface

## TECHNOLOGIES USED
- Python
- Flask
- SQLite
- hashlib
- Regular Expressions
